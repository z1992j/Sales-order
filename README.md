# 销售订单登记

电信业务（宽带）销售订单管理页面，部署在 GitHub Pages。

## 访问地址

https://z1992j.github.io/Sales-order/

## 功能

| 功能 | 说明 |
|------|------|
| 智能录入 | 粘贴下单文本，自动提取姓名/手机/身份证/地址/运营商/套餐/费用 |
| 地址识别 | 根据地址自动识别省市（内置全国主要城市映射） |
| 订单管理 | 新增/编辑/删除/搜索，表格底部合计行 |
| 导出 CSV | 一键下载全部订单 |
| 响应式 | 手机/电脑均可使用 |

## 目录结构

```
.
├── index.html          页面入口
├── orders.json         订单数据
├── .github/workflows/  GitHub Pages 自动部署
└── README.md
```

## 部署

推送代码到 `main` 分支后，GitHub Actions 自动部署到 Pages。

仓库 Settings > Pages 中需选择 **GitHub Actions** 作为 Source。

## 日常使用

将下单信息文本直接发给 Qoder，例如：

```
新增以下广西宽带
姓名：李骞
联系方式：13768515775
身份证：450121199210026017
办理地址：南宁市邕宁区彰泰江景湾5栋1403
办理套餐：电信300M，399元一年
```

Qoder 会自动解析并更新 `orders.json`。
